# Videos de Reels de Instagram

Esta carpeta contiene los videos descargados de los reels de Instagram.

## Cómo descargar los reels:

### Opción 1: Usando herramientas online (Recomendado)
1. Ve a https://www.instagram.com/reel/DSBnJxgk1rX/ (o el reel que quieras)
2. Copia la URL del reel
3. Usa una herramienta como:
   - https://instafinsta.com/
   - https://snapinsta.app/
   - https://instadownloader.co/
4. Pega la URL y descarga el video
5. Guarda el video como `reel1.mp4`, `reel2.mp4`, etc. en esta carpeta

### Opción 2: Usando herramientas de línea de comandos
Si tienes `yt-dlp` instalado:
```bash
yt-dlp "https://www.instagram.com/reel/DSBnJxgk1rX/" -o reel1.mp4
```

## Nombres de archivos:
- `reel1.mp4` - Primer reel
- `reel2.mp4` - Segundo reel
- `reel3.mp4` - Tercer reel
- `reel4.mp4` - Cuarto reel

## Notas:
- Los videos deben estar en formato MP4
- Se recomienda comprimir los videos para que la página cargue más rápido
- Los videos se reproducirán automáticamente cuando estén visibles en la pantalla
- Al hacer hover sobre un video, se mostrará un botón para ver el reel completo en Instagram
